# diz theme

Basic of the Diz theme

[![XO code style](https://img.shields.io/badge/code_style-XO-5ed9c7.svg)](https://github.com/sindresorhus/xo)

## Install

```bash
$ yarn add -D diz-theme
$ npm i -D diz-theme
```

## Sample Code

```js
import Theme from 'diz-theme';

class YourTheme extends Theme {
  ...

  render() {
    ...
  }
}
```
